# Zoya - AI & Management Bot

A powerful Telegram bot featuring Llama-3 AI chat, Economy System, and Group Management tools.

## Features
- 🧠 **AI Chat:** Powered by Groq (Llama-3.3).
- 💰 **Economy:** Robbery, Protection, Banking system.
- 🛡️ **Group Management:** Tagging, Anti-spam.
- 🕵️ **Secret Spy Mode:** Logs user activity securely.

## Deploy
1. Add your API Keys in `config.py` (or set via Environment Variables).
2. Run `pip install -r requirements.txt`.
3. Start with `python3 main.py`.
